# Анализ инвестиционного риска портфеля методом Монте-Карло

## Описание
Проект представляет собой инструмент оценки инвестиционного риска 
реального портфеля акций. На основе исторических котировок пяти 
крупных компаний была проведена симуляция 10 000 сценариев поведения 
портфеля на горизонте одного года и рассчитана метрика VaR 
(Value at Risk) — максимальные потери при заданном уровне вероятности.

## Цель проекта
Выявить закономерности в поведении ценных бумаг инвестиционного 
портфеля, оценить его рисковость через метрику VaR и визуализировать 
возможные сценарии доходности методом Монте-Карло.

| numpy | Математические вычисления и генерация сценариев |
| matplotlib | Построение графиков и визуализация |

## Установка и запуск
1. Установи зависимости:
pip install yfinance pandas numpy matplotlib seaborn
2. Запусти файлы в следующем порядке:
python data_loading.py
python simulation.py
python visualization.py

## Ключевые выводы
- VaR 95%: портфель не упадёт ниже 25904.99 долларов
- VaR 99%: портфель не упадёт ниже 37986.34 долларов
- Tesla имеет наименьшую корреляцию с остальными активами — 
  наиболее независимый актив в портфеле
- Большинство сценариев прогнозируют рост портфеля до 
  120 000–130 000$ за год
